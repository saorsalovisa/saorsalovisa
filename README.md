## 描述

這是一個人際關係心理測驗，旨在幫助你了解自己在人際互動中的態度、反應和行為模式，並進一步提升人際互動的質量。這個網站使用純 HTML、CSS 和 JavaScript 建立，並部署在 GitHub Pages 上。
